# ng2-starter-kit
A starter kit and a sample application for Angular 2


1. Setup our base project:

  # create index.html
  touch index.html

  # create package.json with defaults
  npm init --yes

  # Install lite-server:
  npm install --save-dev lite-server

  # Create an npm script in package.json to start lite-server.
  "scripts": {
    "lite": "lite-server"
  }

  # Run server
  npm run lite
